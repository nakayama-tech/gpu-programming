## セットアップ手順

### 1. リポジトリをチェックアウト

```bash
https://github.com/nakayama-tech/gpu-programming.git
cd gpu-programming
```

### 2. 仮想環境を作成(推奨)

```bash
python -m venv py-pro
```

有効化:

- Windows
  ```bash
  py-pro\Scripts\activate
  ```
- Mac/Linux
  ```bash
  source py-pro/bin/activate
  ```

### 3. 必要なパッケージをインストール

```bash
pip install -r requirements.txt
```

### 4. スクリプトを実行

```bash
python src/your_script.py
```

## ディレクトリ構成

```
your-repo/
├── src/
│   └── your_script.py
├── requirements.txt
└── README.md
```
